# Frozen Paper Evidence

These JSON artefacts are committed evidence for the XCalib paper release.
Public users can independently regenerate the A9 validation file; UTC files are
reference-only unless you already hold the institutional UTC data and weights.

- [`standalone_paper_validation_a9_dataset_r02_s01.json`](standalone_paper_validation_a9_dataset_r02_s01.json) — public A9 validation.
- [`standalone_paper_validation_utc3.json`](standalone_paper_validation_utc3.json) — UTC3 reference validation.
- [`standalone_paper_validation_utc4.json`](standalone_paper_validation_utc4.json) — UTC4 reference validation.
- [`standalone_onnx_validation_utc3.json`](standalone_onnx_validation_utc3.json) — UTC3 ONNX parity reference.
- [`standalone_onnx_validation_utc4.json`](standalone_onnx_validation_utc4.json) — UTC4 ONNX parity reference.
- [`thor_trt_latency.json`](thor_trt_latency.json) — UTC4 Thor TensorRT latency reference.
- [`thor_trt_latency_utc3.json`](thor_trt_latency_utc3.json) — UTC3 Thor TensorRT latency reference.
- [`desktop_rtx5090_supp_speed_table.json`](desktop_rtx5090_supp_speed_table.json) — supplemental desktop throughput table.
